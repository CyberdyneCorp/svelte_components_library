## What's New

- **SankeyChart, ScatterChart, VennDiagram** — Three new chart types for flow analysis, correlation, and set comparison
- **BlockEditor** — Notion-like block-based markdown editor with inline preview, slash commands, and drag reorder
- **PM Charts** — BurndownChart, VelocityChart, CumulativeFlow, AgingWIP for project management
- **MindMap** — Interactive mind mapping with drag-reparent, undo/redo, markdown export
- **MegaMenu + NavBar** — Multi-column dropdown nav and responsive top bar
- **OrgChart** — Hierarchical org chart with detail panels and zoom/pan

---

## Design Principles

**Dark-First** — Every component designed for dark environments with vibrant neon accents.

**Signal over Noise** — Crypto and ML interfaces need clarity: monospace fonts, generous spacing, clear hierarchy.

**Accessible** — WCAG 2.1 AA. Cyan focus rings. Motion respects `prefers-reduced-motion`.

**Composable** — Svelte 5 runes + snippets. CSS custom properties for deep theming.
