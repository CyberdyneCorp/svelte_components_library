## Vision

Cyberdyne builds at the intersection of blockchain infrastructure, AI/ML pipelines, and cutting-edge research platforms. This component library ensures a cohesive, cyberpunk-inspired UI across all products:

- **CyberdyneDAO** -- Web3 terminal platform with NFT-gated access
- **YieldPath** -- AI-powered DeFi life planner
- **Vision Factory** -- Computer vision ML pipeline
- **Terraform Game** -- Blockchain RTS strategy game
- **Research Tools** -- Internal ML and data exploration interfaces
- **MarkdownEditor** -- Rich markdown editing with split preview and Mermaid diagram support
- **MapView** -- Interactive map visualizations for validator nodes and geospatial data

## Design Principles

1. **Dark-First** -- Every component is designed for dark environments with vibrant neon accents referencing the cyberpunk aesthetic.
2. **Signal over Noise** -- In crypto and ML interfaces, data density is high. Components prioritize clarity with monospace fonts, generous spacing, and clear visual hierarchy.
3. **Accessible by Default** -- All components meet WCAG 2.1 AA standards. Focus indicators use the signature cyan glow. Motion respects `prefers-reduced-motion`.
4. **Composable** -- Built with Svelte 5 runes and snippets. Components compose naturally and expose CSS custom properties for deep customization.
