## Test Data

The library ships with a shared test data module at `_testdata/index.ts` containing realistic Cyberdyne-themed data:

- **Crypto market** — Top 10 tokens with prices, market caps, volumes, sparklines
- **Transactions** — Send, receive, swap, approve with various statuses
- **Training metrics** — Loss curves, accuracy, validation loss over 15 epochs
- **Model benchmarks** — MMLU, HumanEval, GSM8K scores
- **Confusion matrix** — 4-class blockchain transaction classification
- **Portfolio timeline** — 12-month portfolio growth with asset allocation breakdown
- **Validator nodes** — 6 Cyberdyne validators with uptime, rewards, status
- **Knowledge graph** — 29 nodes across 5 groups (Infrastructure, Finance, AI/ML, Products, Science)
- **System logs** — 12 entries from ml-service, chain-watcher, yieldpath, vision-factory
- **Deployment terminal** — Full smart contract deployment output

Import in your own stories:
```ts
import { cryptoMarket, trainingMetrics, systemLogs } from "../_testdata/index.js";
```
